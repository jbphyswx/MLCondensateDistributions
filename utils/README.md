# cfSites Data Infrastructure and Usage

This directory contains utilities for reading High-Resolution Large Eddy Simulation (LES) data generated by PyCLES, forced by GCM outputs (cfSites). 

## Data Structure

The cfSite data on the HPC (`/resnick/.../GCMForcedLES/cfSite/`) is output in two main formats:

### 1. 1D Domain-Averaged Profiles (`stats/`)
- **Format**: `Stats.cfSite<SITE>_<MODEL>_<EXP>_<YEARS>.<MONTH>.nc`
- **Dimensions**: `(z, t)` (Height and Time)
- **Time Evolution**: Saves at high frequency. You get the **full time evolution** (e.g., hundreds of time steps for the run).
- **Contents**: Domain-mean fields (`qt`, `ql`, `w`, etc.) and higher-order moments (e.g., variances, fluxes).
- **Usage**: Automatically resolved by `get_stats_path(dir)`. Use Julia's `NCDatasets` to load directly.

### 2. 3D High-Resolution Fields (`fields/` and `fields_combined/`)
- **Format**: `nx × ny × nz` (typically `80 × 80 × 200`)
- **Time Evolution**: 3D data is incredibly large, so **we do not get the full time evolution**. Typically, PyCLES only saves these fields at initialization (`0`) and at the end of the simulation or rare intervals (e.g., `432000`).
- **Raw `fields/` Directory**: 
  - Contains timestamps as subdirectories (e.g., `fields/432000/`).
  - Inside, the domain is split across MPI ranks (`0.nc`, `1.nc`, etc.). Each file has a flattened 1D array (`nl`) and bounding box metadata (`indx_lo_0`, `n_0`, etc.) in the `dims` group that tells you how to reconstruct the 3D chunk.
- **Combined `fields_combined/` Directory**:
  - For some sites and runs, a post-processing script has already stitched the MPI rank files into a standard, single NetCDF file containing contiguous `(nx, ny, nz)` arrays.
  - E.g., `fields_combined/Fields.cfSite13_...432000.nc`.

## Usage Example

```julia
using cfSites
using NCDatasets

# Retrieve path to the output directory
les_dir = get_cfSite_les_dir(10, forcing_model="CNRM-CM5", month=1, experiment="amip")

# 1. 1D Stats (Full time evolution)
stats_file = get_stats_path(les_dir)
ds_stats = NCDataset(stats_file)
qt_mean = ds_stats["qt"][:, :]  # shape: (z, t)
close(ds_stats)

# 2. 3D raw fields dir
fields_time_dir = get_fields_dir(les_dir, timestamp="432000")
# To use raw fields, you must manually reconstruct them across the MPI `.nc` files.

# 3. 3D combined fields (Single Snapshot)
# Note: Throws if the combined file has not been generated for this site!
combined_file = get_combined_field_path(10, forcing_model="CNRM-CM5", month=1, experiment="amip", timestamp="432000")
ds_3d = NCDataset(combined_file)
qt_3d = ds_3d["qt"][:, :, :]  # shape: (nx, ny, nz)
close(ds_3d)
```
